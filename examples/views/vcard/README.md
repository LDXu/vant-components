### 介绍

用于表示状态的小圆点。

支持自定义具体的色值。

### 基础用法

```html
<!--基础-->
<v-card>
  123
</v-card>
```

## API

### Base Props

| 参数   | 说明           | 类型      | 默认值 |
| ------ | -------------- | --------- | ------ |
| status | 类型 success \| error \| default \| processing \| warning | _string_  | `success`    |
| color | 颜色值 | _string_  | _    |
