### 介绍

用来对商品价格数值的小数点前后部分应用不同样式，还支持人民币符号、千位分隔符、设置小数点位数等功能。

### 基础用法

```html
```

```js
export default {
  data() {
    return {
      price: 0
    }
  },
}
```
