# @vcomponetns/Vant-ui

[github pages](https://zehuichan.github.io/vant-components/)

## 前置条件
```js
import Vue from 'vue'

import Vant from 'vant'
import 'vant/lib/index.css'

Vue.use(Vant)
```

### 安装
```sh
npm i vantui-components -S
#or
yarn add vantui-components
```

### 引入组件
```js
// main.js
// 全局引入
import VComponents from 'vantui-components'

Vue.use(VComponents)
```
